About the web map...

This web map is the result of a collaborative research project as part of an Undergraduate Hilldale Research Fellowship. The data compiled in the map comes from Prof. Anna Andrzejewski's research of the city of Madison—specifically its historic suburbs. The data was mostly compliled in a print guidebook to provide content for the Vernacular Architecture Forum's (VAF) annual conference, which was held in Madison in June 2012.

The guidebook possessed such rich information that we felt that it needed multiple platforms. Enter: The web map.

Using open-source geospatial and non-geospatial technologies, such as Leaflet and other JavaScript libraries, this web map can become available to any internet user. Whether navigating a Madison neighborhood first-hand, or virtually navigating on a rainy day, users can access the rich history of Madison's vernacular architecture.

To use the map, click on a neightborhood name in the navigation bar and explore!
